# Changelog

All notable changes to the ETAIQ project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-30

### Added

- Initial monorepo structure for ETAIQ platform
- FastAPI backend with structured logging and configuration
- `GET /health` endpoint returning service status
- Next.js frontend with Dashboard, Prediction, Analytics, History, AI Assistant, and Settings pages
- ML pipeline directory scaffolding
- Docker development environment (backend, frontend, PostgreSQL)
- GitHub Actions CI workflow (lint, test, build)
- Project documentation (spec, architecture, roadmap, contributing)
- Cursor AI development guidance files
- Environment variable template (`.env.example`)
